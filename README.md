Scale LegalAI

AI-powered Legal Intake, Document Summarization & Automated Case Routing

Built for MumbaiHacks 2025 – Agentic AI Track


---

Overview

Scale LegalAI is an end-to-end legal assistance platform combining a React frontend, n8n agentic workflows, local LLMs (Ollama Mistral 7B) and Python text-extraction scripts to help:

✔ Lawyers – process FIRs, complaints, charge sheets, audio statements
✔ Citizens – file domestic/criminal issues through a conversational chatbot

The system automatically:

Extracts text from PDF, DOCX, TXT, and Audio

Converts the content into structured JSON with legal entities

Generates DOCX & PDF reports, and appends data into database.xlsx

Routes client cases via email using Gmail tool-calling


This is 100% offline-capable using local LLMs via Ollama.


---

Key Features

1. Document Intake for Lawyers

Upload legal documents through the Files Dashboard:

PDF / DOCX / TXT uploads

Automatic text extraction via Python

Local LLM processing (Ollama Mistral 7B)

Structured JSON containing:

Parties

Incident summary

Key facts

Evidence

Timeline

Suggested IPC sections

Risk factors


Auto-generation of:

Report.docx

Report.pdf

latest.json

database.xlsx (append mode)




---

2. Client Chatbot (Case Collection Agent)

Through the Chat Dashboard, citizens can:

Explain their issue

Share evidence (WhatsApp msgs, CCTV, etc.)

Provide contact details

Get classified into:

Domestic Case

Criminal Case



The workflow triggers the Gmail tool and sends a structured case email to the appropriate mailbox.


---

3. Automated Email Routing

Based on classification:

Domestic case → domestic.case@gmail.com

Criminal case → criminal.case@gmail.com


Email contains:

Case summary

Evidence description

User contact information

Auto-generated by n8n + Groq + Gmail tool



---

4. Fully Local, Private & Offline-Friendly

No external APIs

Uses Ollama (Mistral 7B)

n8n self-hosted

Perfect for legal confidentiality

All outputs stored locally in data/uploads/reports/

Architecture

Frontend (React)
          ↓
n8n Webhook — File Intake Flow
          ↓
Python Scripts → Extract PDF/DOCX/Audio
          ↓
Ollama (Mistral 7B) → Legal Entity Extraction
          ↓
JS Nodes → JSON Cleanup
          ↓
Python → DOCX / PDF / Excel Generation
          ↓
Frontend receives download links

Client Chat → Groq Chat Model
            ↓
Conversation Memory
            ↓
Classification (domestic / criminal)
            ↓
Gmail Tool — Auto Case Email
